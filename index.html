What is Yog-Sothoth?

Yog-Sothoth is a game engine developed in c ++ for the subject of Game Engine at the CITM-UPC University under the GNU license.

Resource Manager

Manages all resources needed by engine

Object Manager

Manages all game objects, most of the game objects operations must be done through objManager. Game objects are encapuslated to evade possible user errors.

Shader Container

Stores and manages shader info.

ImGUI

Let us draw editor UI.

Import FBX

Importer to manage fbx.

Shaders Pipeline

Shader pipeline is located inside "M_Renderer" file. It orders printable objects by texture to minimize OpenGl texture changing. Accepts 3 types of shader vs, gs and fs. Shaders must be declared into "data/shaders/shaderConfig.json". Shaders can be edited and reloaded from inside engine.
